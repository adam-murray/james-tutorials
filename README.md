## Set Up
```
pyenv virtualenv 3.12 .venv
pyenv activate .venv

pip install bs4
```

## Commiting to git 
* Generate an SSH key 
```
ssh-keygen -t ed25519 -C "your_email@example.com"
```

```
git clone git@github.com:adam-murray/james-tutorials.git
```

```
git checkout -b new_branch_name
```

```
git add file_name
```

```
git commit -m "commit message"

git push 
```
